WordPress se actualizó a la versión 4.8 con una serie de mejoras que, vistas hoy, parecen pequeñas pero tenían bastante sentido para el día a día de cualquier web de contenido.

No fue una versión de ruptura. No cambió la forma de construir páginas ni introdujo un nuevo editor como haría más tarde Gutenberg en WordPress 5.0. Pero sí añadió varias piezas que hacían más fácil editar una web sin depender tanto de plugins externos.

## Los nuevos widgets multimedia

La novedad más visible fue la llegada de tres widgets nuevos:

- Widget de imagen.
- Widget de vídeo.
- Widget de audio.

Hasta entonces, algo tan básico como colocar una imagen o un vídeo en una zona de widgets podía requerir HTML, shortcodes o plugins.

Con WordPress 4.8, el CMS empezaba a reconocer que la edición web tenía que ser más visual y menos técnica.

## El widget de texto también mejoró

Otra mejora importante fue el editor visual dentro del widget de texto.

Esto permitía dar formato al contenido sin tener que escribir HTML a mano. Para usuarios no técnicos, agencias, pequeños negocios y clientes que solo querían actualizar una zona de la web, era un cambio mucho más práctico de lo que parecía.

Ese movimiento encaja con una tendencia que después se hizo más evidente: WordPress necesitaba ser más amigable para competir con constructores visuales, plataformas no-code y sistemas de edición más modernos.

## Menos plugins para tareas sencillas

Uno de los riesgos clásicos de WordPress es instalar demasiados plugins para resolver cosas pequeñas.

Cada plugin añade mantenimiento, posibles conflictos, carga extra y superficie de seguridad. Por eso, cuando el núcleo de WordPress incorpora funciones básicas bien resueltas, la plataforma mejora aunque el cambio no sea espectacular.

Lo mismo ocurre con otros aspectos de una web: optimizar imágenes, cuidar la velocidad o revisar errores SEO no debería depender siempre de añadir capas y más capas. Ya lo comenté en posts como [cómo medir la velocidad web con PageSpeed Insights](/blog/pagespeed/) o [10 errores SEO que debemos evitar](/blog/errores-seo/).

## Qué lectura deja WordPress 4.8

WordPress 4.8 fue una actualización discreta, pero coherente: acercar la edición de contenido a usuarios normales.

Hoy damos por hecho que un CMS permita arrastrar bloques, insertar imágenes, incrustar vídeos o editar textos con una interfaz visual. En 2017, estas pequeñas mejoras eran parte del camino hacia esa experiencia.
