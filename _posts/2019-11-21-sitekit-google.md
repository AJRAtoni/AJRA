**Site Kit by Google** es el plugin oficial de Google para conectar servicios de medición y rendimiento dentro de WordPress.

Su objetivo es facilitar la configuración de herramientas como Analytics, Search Console y PageSpeed Insights sin tocar código.

## Qué aporta

Una vez instalado y configurado, Site Kit permite ver datos directamente desde el escritorio de WordPress:

- Tráfico.
- Consultas de búsqueda.
- Rendimiento.
- Métricas de PageSpeed.
- Información básica de monetización si se conectan servicios compatibles.

Para muchos usuarios, la ventaja está en centralizar datos sin abrir varias pestañas.

## Cuándo tiene sentido

Site Kit puede ser útil si gestionas una web WordPress y quieres una configuración rápida.

La instalación se hace desde el [repositorio oficial de WordPress](https://wordpress.org/plugins/google-site-kit/) y el asistente guía la conexión con la cuenta de Google.

Personalmente, sigo prefiriendo entender qué se está instalando y por qué. Medir no es solo poner un plugin.

Si usas Analytics, Search Console o PageSpeed, conviene saber qué pregunta responde cada herramienta.
