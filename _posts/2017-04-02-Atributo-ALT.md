El **atributo alt** es uno de esos detalles pequeños que dicen mucho de la calidad de una web.

Sirve para describir una imagen cuando esta no puede verse o cuando una persona usa tecnologías de asistencia, como lectores de pantalla.

También ayuda a los buscadores a entender mejor el contenido visual de una página.

## Qué es el atributo alt

El atributo `alt` es el texto alternativo de una imagen en HTML.

Un ejemplo sencillo:

```html
<img src="/images/ejemplo.jpg" alt="Persona revisando métricas de una campaña de marketing digital">
```

Si la imagen no carga, el navegador puede mostrar ese texto. Si una persona usa un lector de pantalla, ese texto ayuda a entender qué aporta la imagen.

## Para qué sirve

El atributo alt cumple varias funciones:

- Mejora la accesibilidad.
- Ayuda a interpretar imágenes si no cargan.
- Aporta contexto semántico a la página.
- Puede ayudar al SEO de imágenes.
- Mejora la calidad general del contenido.

Antes se hablaba mucho del alt como un truco SEO. Hoy prefiero verlo primero como una práctica de accesibilidad y calidad.

## Cómo escribir buenos textos alt

Un buen alt debe describir la imagen de forma útil y natural.

Recomendaciones:

- Describe lo importante de la imagen.
- No rellenes con palabras clave sin sentido.
- Sé concreto, pero no escribas un párrafo eterno.
- Si la imagen es decorativa, puede tener alt vacío.
- Si la imagen contiene texto importante, inclúyelo o explícalo en el contenido.

Mal ejemplo:

```html
<img src="zapatos.jpg" alt="zapatos zapatos comprar zapatos baratos zapatos online">
```

Buen ejemplo:

```html
<img src="zapatos.jpg" alt="Zapatillas deportivas rojas sobre fondo blanco">
```

## Alt, SEO y experiencia de usuario

El alt puede ayudar al posicionamiento, pero no debería escribirse solo para Google.

Una web bien cuidada combina SEO, rendimiento, accesibilidad y contenido claro. Por eso este tema conecta con otros aspectos técnicos como la [velocidad de carga](/blog/pagespeed/), la adaptación [mobile friendly](/blog/google-mobile-friendly/) o los [errores SEO](/blog/errores-seo/).

Si estás trabajando ecommerce, el alt en imágenes de producto también puede ayudar a que el catálogo sea más comprensible para usuarios y buscadores.

## Conclusión

El atributo alt es un detalle pequeño, pero importante.

No arregla una mala estrategia SEO por sí solo, pero forma parte de una web mejor hecha: más accesible, más clara y más fácil de entender.
