Empiezo una serie de recursos sencillos para crear, gestionar o hacer crecer un proyecto.

El primero es **I Love PDF**, una herramienta online para resolver tareas habituales con archivos PDF.

<img src="/images/blog/270-1.webp" alt="Interfaz de I Love PDF con herramientas para documentos PDF" />
</figure>

## Qué permite hacer

Desde la web de I Love PDF puedes:

- Unir PDFs.
- Dividir PDFs.
- Convertir PDFs a Word, Excel o PowerPoint.
- Convertir una web a PDF.
- Editar PDFs online.
- Añadir marcas de agua.
- Proteger PDFs con contraseña.
- Firmar PDFs.

Para tareas rápidas, basta con entrar, subir el archivo y elegir la acción.

## Cuándo usarlo

Es útil cuando necesitas salir del paso sin instalar una app pesada.

Para documentos sensibles conviene pensar dos veces dónde subes el archivo, pero para muchos casos cotidianos funciona muy bien.

También complementa flujos del iPad, como [unir PDFs desde iPadOS](/blog/ipad-unir-pdf/), [firmar PDFs](/blog/ipad-firmar-pdf/) o convertir un [PDF en JPEG con Atajos](/blog/ipad-07/).

Puedes probarlo en [ilovepdf.com](https://www.ilovepdf.com).
